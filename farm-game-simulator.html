Farm Game/Simulator

I am currently working on this project as part of a group of 5 for my objects and design course CS 2340. We are creating a farm game with features like harvesting, inventory, marketing, and a bank.

In the process of creating this project, we are using the unified process with artifacts like system sequence diagrams, robustness diagrams, design class diagrams, domain models, and package diagrams.

This projects involves use of JavaFX, TestFX, and GitHub.

Farm Game/Simulator

I recently finished this project as part of a group of 5 for my objects and design course, CS 2340. We created a farm game with features like harvesting, irrigation, random events such as locusts, inventory, a market, and a bank.

In the process of creating this project, we used the unified process with artifacts like system sequence diagrams, robustness diagrams, design class diagrams, domain models, and package diagrams. We also got to learn about and apply SOLID and GRASP.

This projects involves use of JavaFX, TestFX, and GitHub.

Here are some screenshots of our final product. Note that the point of this project was to learn about JavaFX, software design, and the unified process, not aesthetics or an appealing UI (hence the gradient on the welcome page and some other questionable design choices).
